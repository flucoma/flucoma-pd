fluid.bufinfo

Query fluid.multiarray

Description

An abstraction for querying the dimensions of a fluid.multiarray

Discussion

This object is designed for testing the validity, the size in frames and the number of channels of a fluid.multiarray, either when created via the clone object, or programmatically with automatically generated 'buffers'.

fluid.bufinfo is part of the Fluid Decomposition Toolkit of the FluCoMa project. For more explanations, learning material, and discussions on its musicianly uses, visit flucoma.org.

Inlets

  • 1 [anything]

    See below for the various messages.

Outlets

  • 1 [list]

    A list of 2 numbers: the size in frames and the number of 'channels'.

Arguments

Attributes

source [symbol] (default: )

The 'multiarray' / 'buffer' to report on.

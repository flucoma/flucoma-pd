fluid.concataudiofiles

A utility to concatenate audio files

Description

Given a list of valid audio file paths, concatenates them into a single array.

Discussion

This object takes a list of paths to audio files, loads them one after the other into a fluid.multiarray, and fills a text with the filenames, start and end frames, sampling rate and channel count.

fluid.concataudiofiles is part of the Fluid Decomposition Toolkit of the FluCoMa project. For more explanations, learning material, and discussions on its musicianly uses, visit flucoma.org.

Inlets

    Bang triggers the process.

Outlets

    Bang when done.

Messages

list

Arguments: path1 (symbol) path2 (symbol) pathN (symbol)

Trigger concatenation of the audio files listed.

Argument details:
  • path1:

    The complete path to the first audiofile to load.

  • path2:

    The path to the second file to load.

  • pathN:

    The path to the Nth file to load.

fluid.ds2text

Convert FluCoMa DataSet into text

Description

A utility object to convert a fluid.dataset into text.

Discussion

This is an object for conveniently converting a fluid.dataset into text.

fluid.ds2text is part of the Fluid Decomposition Toolkit of the FluCoMa project. For more explanations, learning material, and discussions on its musicianly uses, visit flucoma.org.

Inlets

  • 1 [bang]

    Bang triggers the process.

  • 2 [symbol]

    Change the fluid.dataset that is converted to text.

Outlets

  • 1 [bang]

    Bang when done.

Arguments

dataset [symbol]

(default: none)
A source dataset

text [symbol]

Optional
(default: none)
A destination text object

fluid.plotter

Plot data in a two-dimensional graph.

Description

A GUI object that plots data in a two-dimensional graph.

Discussion

The fluid.plotter object can be used as a versatile solution for plotting points in a two-dimensional graph. It is designed to easily accept the contents of a fluid.dataset via a [text] object as coordinates within the graph as well as the contents of a fluid.labelset via a [text] object as colours for points. There is also an 'advanced' interface that allows the user to specify the coordinates, sizes and colours of the points in a more flexible and bespoke manner. See the help file for more information on this.

fluid.plotter is part of the Fluid Decomposition Toolkit of the FluCoMa project. For more explanations, learning material, and discussions on its musicianly uses, visit flucoma.org.

Messages

range

Arguments: min (number) max (number)

Set the x- and y-axis ranges.

Argument details:
  • min:

    The minimum value for the x-axis and y-axis

  • max:

    The maximum value for the x-axis and y-axis

xrange

Arguments: min (number) max (number)

Set the x-axis range.

Argument details:
  • min:

    The minimum value for the x-axis

  • max:

    The maximum value for the x-axis

yrange

Arguments: min (number) max (number)

Set the y-axis range.

Argument details:
  • min:

    The minimum value for the y-axis

  • max:

    The maximum value for the y-axis

highlight

Arguments: identifiers (list)

Emphasises the given identifier's points in the plot

Argument details:
  • identifiers:

    Any number of identifiers as a list of symbols.

clear

Arguments:

Clears the graph and all associated data and settings.

Argument details:

setpoints

Arguments: text (symbol)

Set the position for points by their data in a [text] object, usually retrieved from a fluid.dataset.

Argument details:
  • text:

    The name of a text object containing data from a fluid.dataset.

setlabels

Arguments: text (symbol)

Set the labels for each point according to their labels in a [text] object, usually retrieved from a fluid.labelset.

Argument details:
  • text:

    The name of a text object containing labels from a fluid.labelset. This message must enter the right inlet.

pointsize

Arguments: text (symbol) size (number)

Set the size of a single point.

Argument details:
  • text:

    The identifier of the point to resize.

  • size:

    The radius of the point to draw in pixel.

pointsizescale

Arguments: size (number)

Set the size scaling factor of all points.

Argument details:
  • size:

    The scaling factor of the size in pixel.

pointcolor

Arguments: text (symbol) colour (number)

Set the colour of a single point.

Argument details:
  • text:

    The identifier of the point to modify.

  • colour:

    The class (0 to 7) or Pd colour value (999 -> RGB) for the point.

setpoint

Arguments: text (symbol) x-pos (number) y-pos (number) size (number) colour (number)

Creates or overwrites a single point.

Argument details:
  • text:

    The identifier of the point to create or modify.

  • x-pos:

    The position of the point on the x-axis.

  • y-pos:

    The position of the point on the y-axis.

  • size:

    The radius of the point to draw in pixel.

  • colour:

    The class (0 to 7) or Pd colour value (999 -> RGB) for the point.

fluid.s2f

Convert a symbol into a float

Description

Convert a floating point value expressed as a symbol ('string') and output it as a float.

Discussion

This object does the opposite of sending a float into [makefilename %f]. Note that the symbol needs to be enclosed in double quotes for s2f to work.

fluid.s2f is part of the Fluid Decomposition Toolkit of the FluCoMa project. For more explanations, learning material, and discussions on its musicianly uses, visit flucoma.org.

Inlets

  • 1 [symbol]

    The symbol to convert to a float.

Outlets

  • 1 [float]

    The converted input as float.

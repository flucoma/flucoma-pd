fluid.multiarray

Multichannel arrays

Description

An abstraction that assists in the management and creation of multichannel arrays.

Discussion

This abstraction allows the creation of multichannel arrays. Note that it requires invoking via the [clone] object to work, where the number of instances is the number of 'channels'.

fluid.multiarray is part of the Fluid Decomposition Toolkit of the FluCoMa project. For more explanations, learning material, and discussions on its musicianly uses, visit flucoma.org.

Inlets

  • 1 [anything]

    Messages to all the underlying instances of arrays in the 'multiarray'.

Outlets

Arguments

name [symbol]

The name of the multiarray. Each component array is using this name, followed by the 'channe' number. For instance, name-0 is the zeroth channel of name. The usual send-to-array syntax via ;name will address all the channels at once.

size [int]

The number of elements in each channel array.

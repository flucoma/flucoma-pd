fluid.waveform

Plot fluid.multiarrays in various ways.

Description

A GUI object that plots data from fluid.multiarrays of audio, features, images and indices.

Discussion

The fluid.waveform object can be used as a versatile solution for plotting arrays and multiarrays. It plots multichannel audio as waveform. It plots time-series of features as lines. It plots images of 2D matrices, i.e. spectrograms. It plots indices as markers. See the helpfile for various examples.

fluid.waveform is part of the Fluid Decomposition Toolkit of the FluCoMa project. For more explanations, learning material, and discussions on its musicianly uses, visit flucoma.org.

Messages

waveform

Arguments: buffer (symbol)

Set the source 'buffer' for the waveform to draw. This can be a multichannel audiofile and will be normalised across channels.

Argument details:
  • buffer:

    The name of the "fluid.multiarray" to draw as a waveform.

features

Arguments: buffer (symbol)

Set the source 'buffer' for the features to draw. Each feature is normalised individually.

Argument details:
  • buffer:

    The name of the "fluid.multiarray" to draw as lines of features.

image

Arguments: buffer (symbol)

Set the source 'buffer' for the image matrix to draw. The minimum is mapped to dark and the maximum is mapped to bright. This is sometimes referred to as a heat map.

Argument details:
  • buffer:

    The name of the "fluid.multiarray" to draw as a 2d matrix.

indices

Arguments: buffer (symbol)

Set the source 'buffer' for the indices to draw. This needs to happen after the waveform is set in order to define the timeframe of the markers.

Argument details:
  • buffer:

    The name of the array, or multiarray, containing indices to draw as markers.

imagelogflag

Arguments: flag (int)

Set how the full range of values in the 2d matrix is mapped to color. 0 is linear, 1 is logarithmic.

Argument details:
  • flag:

    The flag: 0 is linear, 1 is logarithmic.

linewidth

Arguments: width (int)

Set the line width for both the markers of indices and the lines of features.

Argument details:
  • width:

    The width of the line.

waveformcolor

Arguments: colour (symbol)

Set the colour of the waveform. This is the new Pd colour format #rrggbb where rr,gg,bb are hexadecimal for each RGB component.

Argument details:
  • colour:

    The colour in #RRGGBB format.

featurescolor

Arguments: colour (symbol)

Set the colour of the features lines. This is the new Pd colour format #rrggbb where rr,gg,bb are hexadecimal for each RGB component.

Argument details:
  • colour:

    The colour in #RRGGBB format.

imagecolorscheme

Arguments: scheme (int)

Set the perceptually linear color scheme used to draw the matrix values. See the ColorCET website for more information.

Argument details:
  • scheme:

    The colour scheme. 0 is CET-L02, 1 is CET-L16.

indicescolor

Arguments: colour (symbol)

Set the colour of the marker line drawn for indices. This is the new Pd colour format #rrggbb where rr,gg,bb are hexadecimal for each RGB component.

Argument details:
  • colour:

    The colour in #RRGGBB format.
